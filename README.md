# VendingMachine(自動販売機)

## Overview
街中に点在する自動販売機をモデリングして、コード化したもの。  

## Description
画面上で商品を選択し、購入する流れを実現。DDDで作成。  
入金時の硬貨や紙幣の計算、お釣りの計算、決済の成否判定など。  
在庫管理機能など、目に見えないところの機能を実装。  

## Requirement
SpringBoot 2.1.3.RELEASE  
Thymeleaf 3.0.0.RELEASE
